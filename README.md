## 简介

edu：education，在线课程网站

## 主要功能

*  课程管理：创建课程、添加课时、添加课程资料
*  用户管理：角色管理、权限分配
*  个人中心：学习进度、课程收藏、文章收藏
*  文件管理：上传、删除文件，文件组管理
*  数据管理：数据备份、恢复
